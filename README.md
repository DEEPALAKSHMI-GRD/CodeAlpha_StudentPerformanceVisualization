# Student Performance Data Visualization

This project is part of the CodeAlpha Data Analytics Internship.

## Objective
To analyze and visualize student performance data using Python libraries like Matplotlib and Seaborn.

## Dataset
The dataset contains student scores across different subjects and their results.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Visualizations
- Average Score by Subject
- Pass vs Fail Distribution
- Score Distribution

## Outcome
The visualizations help understand performance trends and support data-driven decisions.
